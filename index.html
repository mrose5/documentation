AFLOW REST API

The AFLOW REST API enables users to search and download results from the AFLOW database. For more details on how to download the data, please consult the original publication: R. H. Taylor, F. Rose, C. Toher, O. Levy, K. Yang, M. B. Nardelli, S. Curtarolo, A RESTful API for exchanging materials data in the AFLOWLIB.org consortium, Comp. Mat. Sci. 93, 178-192 (2014). [DOI: 10.1016/j.commatsci.2014.05.014]
